# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) — the full text is at that link.

## TL;DR

Be kind. Assume good faith. Disagree without belittling. We're here to build threat-intel tooling that operators can actually trust, and that takes a working group where people aren't afraid to ship rough first drafts or admit they were wrong.

Not OK: harassment, personal attacks, demeaning comments (about anything — code, taste, identity, background), unwelcome sexual attention, doxxing, sustained disruption of discussions.

## Reporting

If you experience or witness behaviour that violates this, email **[rinjanianalytics@gmail.com](mailto:rinjanianalytics@gmail.com)** — please don't file a public issue. We will:

- Acknowledge within 2 business days
- Investigate confidentially
- Take action proportional to the severity (which may include warning, temporary ban, or permanent removal from project spaces)

We will not retaliate against reporters acting in good faith.

## Scope

This applies in all project spaces — issues, PRs, discussions, code comments, the project's social channels, and any in-person events held under the project's name.
